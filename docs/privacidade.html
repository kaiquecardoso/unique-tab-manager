Política de Privacidade — One Tab Manager

Última atualização: maio de 2026

O One Tab Manager é uma extensão para Google Chrome que permite salvar e gerenciar abas do navegador em uma lista local. Esta política descreve como os dados são tratados na versão publicada na Chrome Web Store.

1. Dados tratados

A extensão pode armazenar apenas no seu dispositivo:

  • URLs e títulos das abas que você escolhe salvar;
  • tags, favoritos e status de “visualizado” que você define;
  • itens na lixeira e backups importados por você;
  • preferências da extensão (tema, layout, filtros, sites excluídos).

Esses dados correspondem ao tipo “histórico da web” declarado na Chrome Web Store, pois incluem páginas e títulos que você decidiu guardar — não um rastreamento automático de toda a sua navegação.

2. O que não coletamos

  • Não exigimos conta, login ou e-mail;
  • Não enviamos seus dados a servidores do desenvolvedor;
  • Não vendemos nem compartilhamos dados com terceiros para marketing;
  • Não usamos dados para crédito, publicidade personalizada ou perfilamento.

3. Onde os dados ficam

Tudo é gravado no armazenamento local do navegador (chrome.storage.local). Apenas você, neste perfil do Chrome, tem acesso a essas informações, salvo se você exportar um backup manualmente ou usar recursos de sincronização do próprio Google Chrome entre dispositivos.

4. Permissões e uso limitado

A extensão usa permissões do Chrome somente para o propósito de salvar e gerenciar abas (por exemplo: ler URL/título da guia ativa, fechar guias salvas, menu de contexto e página de opções). Scripts de conteúdo podem exibir confirmações na página quando você salva um link ou quando há duplicata.

5. Serviços de terceiros (somente na interface)

Para exibir ícones de sites, a interface pode carregar imagens de favicon de provedores públicos (por exemplo, Google Favicon). Isso não envia seus dados salvos ao desenvolvedor; é equivalente a carregar uma imagem em uma página web. Em páginas do painel LivePix, um botão opcional permite salvar links de mensagens para sua lista local, somente quando você clica nele.

6. Código remoto

Toda a lógica da extensão está incluída no pacote instalado. Não executamos JavaScript baixado de servidores externos. Requisições de rede podem ocorrer apenas para obter o título HTML de um link que você está salvando, sem executar código remoto.

7. Seus direitos

Você pode, a qualquer momento:

  • excluir abas, grupos ou esvaziar a lixeira na página de opções;
  • exportar ou importar backup em JSON;
  • desinstalar a extensão, o que remove os dados locais associados a ela.

8. Alterações

Podemos atualizar esta política se o funcionamento da extensão mudar. A data no topo desta página será revisada quando isso ocorrer.
